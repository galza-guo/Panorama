# 介绍

欢迎使用
**Panorama**，这是一款美观、隐私且开源的投资追踪器，以此为基础特别增强了对中国市场的支持。

Panorama 是 [Wealthfolio](https://wealthfolio.app)
的透明分叉版本，专为亚洲市场和额外的资产类别提供了专门的增强功能。

## 为什么选择 Panorama？

Panorama 建立在 Wealthfolio 坚实的基础之上，增强了对亚洲市场的支持和额外的资产类别，同时保持了隐私和简洁的核心原则。

### 🚀 Panorama 独家功能

- **🇨🇳 中国股市支持**：原生支持 **A股**，提供来自东方财富的实时报价。
- **📈 全面的基金数据**：支持 **公募基金** 和
  **场外基金**，数据由东方财富和天天基金强力驱动。
- **🇭🇰 香港强积金 (MPF)**：专门追踪 **强积金** 资产，并提供基金配置可视化。
- **🛡️ 保险追踪**：全面支持保险保单，包括人寿保险、投连险 (ILAS) 和医疗保险。

### ✨ 主要功能 (继承自 Wealthfolio)

- **桌面优先**：专为桌面环境优化（Windows, macOS, Linux）。绝无移动端妥协。
- **本地 & 隐私**：您的数据存储在本地设备的 SQLite 数据库中。无服务器，无追踪，无订阅。
- **开源**：代码在 [GitHub](https://github.com/galza-guo/Panorama) 上完全透明。
- **全面追踪**：跨账户追踪股票、ETF、加密货币和自定义资产。
- **绩效指标**：通过时间加权收益率和基准对比来监控您的投资组合表现。
- **🧩 可扩展**：强大的插件系统，用于自定义功能。
