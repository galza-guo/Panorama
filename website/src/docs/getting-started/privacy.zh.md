# 隐私与安全

在 Panorama，我们相信您的财务数据只属于您自己。

## 本地优先设计

Panorama 遵循“本地优先”理念。这意味着：

- **无云端存储**：您的数据从未上传到我们的服务器（因为我们根本没有服务器！）。
- **本地数据库**：您的所有交易、账户详情和设置都存储在您自己电脑上的加密 SQLite 数据库中。
- **完全掌控**：您可以随时备份、导出或删除您的数据。

## 数据使用

- **市场数据**：为了提供实时价格，Panorama 会连接到公共市场数据提供商（如 Yahoo
  Finance）。查询直接从您的电脑发出。
- **更新**：应用会定期检查 GitHub 以获取新版本。

我们不会追踪您的使用情况，不会出售您的数据，也不会以任何方式利用您的财务信息获利。
