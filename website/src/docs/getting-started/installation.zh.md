# 安装

安装 Panorama 非常简单。由于它是桌面应用程序，您只需下载并安装即可。

## 安装步骤

### macOS

1.  前往
    [Releases 页面](https://github.com/galza-guo/Panorama/releases/latest)。
2.  下载适用于您架构的 `.dmg` 文件（Apple Silicon 为 `aarch64`，Intel 为
    `x64`）。
3.  打开 `.dmg` 文件并将 Panorama 拖入“应用程序”文件夹。

### Windows

1.  前往
    [Releases 页面](https://github.com/galza-guo/Panorama/releases/latest)。
2.  下载 `.exe` 安装程序。
3.  运行安装程序并按照提示操作。

### Linux

1.  前往
    [Releases 页面](https://github.com/galza-guo/Panorama/releases/latest)。
2.  下载 `.AppImage` 或 `.deb` 文件。
3.  运行应用程序。

## 初始设置

首次启动 Panorama 时，您将通过一个简单的设置过程：

1.  **选择基础货币**：选择您投资组合的主要货币。
2.  **添加账户**：创建您的第一个账户（例如，“主经纪账户”、“储蓄账户”）。
3.  **导入或添加资产**：您可以手动添加资产或通过 CSV 导入交易。

设置完成后，您将进入仪表盘。
