# Getting Started

Getting up and running with Panorama is simple. Since it is a desktop
application, you just need to download and install it.

## Installation

### macOS

1.  Go to the
    [Releases page](https://github.com/galza-guo/Panorama/releases/latest).
2.  Download the `.dmg` file for your architecture (Apple Silicon `aarch64` or
    Intel `x64`).
3.  Open the `.dmg` file and drag Panorama into your Applications folder.

### Windows

1.  Go to the
    [Releases page](https://github.com/galza-guo/Panorama/releases/latest).
2.  Download the `.exe` installer.
3.  Run the installer and follow the prompts.

### Linux

1.  Go to the
    [Releases page](https://github.com/galza-guo/Panorama/releases/latest).
2.  Download the `.AppImage` or `.deb` file.
3.  Run the application.

## Initial Setup

When you first launch Panorama, you will be guided through a simple setup
process:

1.  **Select Base Currency**: Choose the primary currency for your portfolio.
2.  **Add Accounts**: Create your first account (e.g., "Main Brokerage",
    "Savings").
3.  **Import or Add Assets**: You can manually add assets or import transactions
    via CSV.

Once setup is complete, you will be taken to your Dashboard.
