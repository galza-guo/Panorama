# 市场数据 & 外汇

Panorama 提供强大的多币种和全球市场支持。

## 支持的资产

- **股票 & ETF**：美国、英国、欧洲、亚洲（A股、港股）等。
- **加密货币**：比特币、以太坊和数千种山寨币。
- **货币**：外汇对。

## 市场数据来源

Panorama 从多个公开提供商获取实时报价：

- **全球市场**：Yahoo Finance、Alpha Vantage 等开放 API。
- **中国 A 股**：由东方财富强力驱动 - 无需 API 密钥。
- **中国基金**：净值数据来自天天基金。
- **汇率**：来自 Open Exchange Rates (OXR) 的可靠外汇数据。

所有市场数据查询直接从您的电脑发出到这些提供商，不经过中间服务器。

## 货币处理

Panorama 在四个层面上自动处理货币转换：

1.  **基础货币**：整个投资组合仪表盘的主要货币（例如，CNY, USD）。
2.  **账户货币**：特定账户的计价货币（例如，EUR 银行账户）。
3.  **资产货币**：资产交易的货币（例如，TSLA 以 USD 交易）。
4.  **活动货币**：特定交易使用的货币。

系统使用历史汇率来正确计算过去交易的价值。
