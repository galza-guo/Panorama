# 活动类型

Panorama 将资金变动归类为特定的活动类型，以准确追踪您的投资组合表现和税务影响。

| 类型             | 描述                           | 现金影响 | 资产影响 |
| :--------------- | :----------------------------- | :------- | :------- |
| **BUY**          | 买入资产                       | 减少     | 增加     |
| **SELL**         | 卖出资产                       | 增加     | 减少     |
| **DIVIDEND**     | 从资产收到的现金（如股票股息） | 增加     | 无       |
| **INTEREST**     | 现金或债券的利息               | 增加     | 无       |
| **DEPOSIT**      | 向账户存入现金                 | 增加     | 无       |
| **WITHDRAWAL**   | 从账户提取现金                 | 减少     | 无       |
| **FEE**          | 交易或维护费用                 | 减少     | 无       |
| **TAX**          | 支付税款                       | 减少     | 无       |
| **TRANSFER_IN**  | 移入资产/现金                  | 增加     | 增加     |
| **TRANSFER_OUT** | 移出资产/现金                  | 减少     | 减少     |
| **SPLIT**        | 股票拆分（如 2拆1）            | 无       | 更改数量 |

## 快速入门速查表

- **从零开始**：使用 **DEPOSIT** 为您的账户注资，然后 **BUY** 购买资产。
- **导入历史**：如果从经纪商导入，请确保您的 CSV 正确映射了“Buy”和“Sell”。
- **收入**：股票/ETF 的派息使用 **DIVIDEND**，银行利息使用 **INTEREST**。
