# 绩效指标

Panorama 使用行业标准指标帮助您了解投资组合的表现。

## 时间加权收益率 (TWR)

TWR 衡量投资组合的复合增长率。它消除了资金流入流出的扭曲影响。这是将您的表现与基准（如标准普尔 500 指数）进行比较的最佳指标，因为它将您的投资决策与存款/取款时机隔离开来。

## 资金加权收益率 (MWR / IRR)

MWR（通常称为内部收益率）考虑了现金流的时机和规模。它反映了您实际获得的资金回报。如果您能很好地把握市场时机（低买高卖），您的 MWR 将高于 TWR。

## 简单回报率

价值的绝对百分比变化。\`(当前价值 - 成本基础) / 成本基础\`

## 波动率 & 最大回撤

- **波动率**：衡量投资组合价值的波动程度。
- **最大回撤**：从峰值到谷底的最大百分比跌幅。它有助于您了解所承担的风险。
