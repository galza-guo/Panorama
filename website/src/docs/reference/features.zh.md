# 功能

Panorama 自带强大的工具，帮助您管理财富，并针对亚洲市场进行了特别增强。

## 仪表盘

您的财务指挥中心。

- **净资产**：实时计算您的总财富。
- **绩效**：显示投资组合随时间增长的可视化图表。
- **配置**：按类型、行业或自定义标签细分您的资产。

## 活动追踪

记录您资金的每一次变动。

- **交易**：买入、卖出、存款、取款、股息、利息。
- **CSV 导入**：轻松导入经纪商的历史记录。
- **历史**：可搜索的所有过去活动列表。

## 目标与限制

保持正轨。

- **财务目标**：设定目标净资产或储蓄目标，并追踪进度。
- **供款限制**：监控您对税收优惠账户（如 IRA 或 ISA）的供款，以避免罚款。

## 市场数据

- **全球支持**：追踪来自全球主要交易所（美国、欧盟、亚洲）的资产。
- **加密货币**：集成对主要加密货币的支持。
- **外汇**：为多币种投资组合提供自动货币转换。

---

## 🚀 Panorama 独家功能

### 🇨🇳 中国市场支持

- **A股追踪**：原生支持上海 (SH) 和深圳 (SZ) 上市股票，提供实时报价。
- **代码格式**：标准格式 `600001.SH` / `000001.SZ`，易于识别。
- **数据源**：由东方财富强力驱动 - 无需 API 密钥。

### 📈 基金管理

- **公募基金**：追踪中国公募基金，净值更新来自天天基金。
- **场外基金**：支持场外基金申购。
- **代码格式**：`161039.FUND` 格式用于基金识别。

### 🇭🇰 香港强积金 (MPF)

专门的强积金追踪支持：

- **基金配置**：可视化您的强积金投资组合配置。
- **受托人管理**：跨不同受托人追踪强积金。
- **绩效监控**：监控强积金基金随时间的表现。

### 🛡️ 保险追踪

全面的保险保单管理：

- **保单类型**：支持人寿保险、投连险 (ILAS) 和医疗保险。
- **保单详情**：追踪保单号码、保证价值和到期日。
- **保费追踪**：监控保费支付和到期日。

## 可扩展性

Panorama 支持强大的插件系统：

- **自定义页面**：创建您自己的仪表盘和视图。
- **数据访问**：完全访问投资组合、账户和市场数据。
- **UI 集成**：添加导航项和组件。

更多信息请参见 [插件概览](../addons/overview.md)。
