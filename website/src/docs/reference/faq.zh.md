# 常见问题

### Panorama 是免费的吗？

是的！Panorama 是完全免费和开源的软件。您可以下载、使用甚至修改它，无需支付一分钱。

### 我的数据安全吗？

绝对安全。由于 Panorama 是本地优先的应用程序，您的数据从未离开过您的电脑。它存储在只有您能访问的本地数据库中。

### 我可以从 Wealthfolio 导入数据吗？

由于 Panorama 是 Wealthfolio 的分叉，数据库结构在很大程度上是兼容的。您应该能够导入备份文件，但我们建议导入后仔细检查数据。

### 有移动端应用吗？

目前，Panorama 100% 专注于 **桌面** 体验（macOS, Windows, Linux），将所有资源投入到打造最好的桌面投资追踪器。

### 如何更新 Panorama？

Panorama 会在启动时自动检查更新。当 GitHub 上有新版本时，您会收到下载和安装的通知。

### 我的数据存储在哪里？

- **macOS**: `~/Library/Application Support/Panorama/`
- **Windows**: `C:\Users\<User>\AppData\Roaming\Panorama\`
- **Linux**: `~/.config/panorama/`
