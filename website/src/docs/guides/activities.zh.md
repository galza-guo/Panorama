# 管理活动

活动是 Panorama 的核心。每一笔交易、股息或存款都是一个“活动”。

## 添加活动

您可以通过两种方式添加活动：

1.  **手动输入**：点击 “**+**” 按钮，选择活动类型（买入、卖出等），然后填写详细信息。
2.  **CSV 导入**：从您的经纪商导入历史文件。

## CSV 导入指南

Panorama 支持通过 CSV 格式导入。您的 CSV 文件理想情况下应包含以下列：

- `Date` (日期，格式 YYYY-MM-DD)
- `Type` (类型，如 Buy, Sell, Deposit 等)
- `Symbol` (代码，如 AAPL)
- `Quantity` (数量)
- `Price` (价格)
- `Fee` (费用，可选)
- `Currency` (货币，可选，默认为账户货币)

## 编辑与删除

您可以从 **活动**
标签页编辑或删除任何活动。点击某一行以打开详情面板，您可以修改数值或完全删除该条目。
