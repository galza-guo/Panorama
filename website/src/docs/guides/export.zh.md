# 导出与备份

由于 Panorama 是本地优先的，备份数据是您的责任。我们让这一切变得简单。

## 备份 (SQL)

您可以将整个数据库导出为单个文件。

- 前往 **设置 > 导出**。
- 选择 **数据库备份**。
- 将文件 (`panorama-backup.sqlite`) 保存到安全位置。
- **恢复**：您可以在任何新安装的 Panorama 上恢复此文件。

## 导出 (CSV/JSON)

如果您想在 Excel 或其他工具中使用您的数据：

- **交易 CSV**：导出每一笔活动。
- **持仓 CSV**：导出当前的投资组合快照。
- **JSON**：导出整个投资组合状态的结构化 JSON 文件。
