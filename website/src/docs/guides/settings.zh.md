# 设置

自定义 Panorama 以满足您的需求。

## 常规

- **基础货币**：这是您全局“净资产”使用的货币。更改此选项将使用历史汇率将您的所有历史数据转换为新货币。
- **格式**：选择您喜欢的日期格式 (DD/MM/YYYY 或 MM/DD/YYYY) 和数字格式 (1,000.00 或 1.000,00)。

## 外观

- **主题**：在 **浅色**、**深色** 或 **系统** 模式之间切换。
- **隐私模式**：模糊仪表盘上的数值（在公共场合使用 Panorama 时很有用）。

## 数据管理

- **删除所有数据**：**警告**：这将永久清除您的本地数据库。无法撤销。
