# 管理账户

账户代表您持有资产的地方（例如，“Chase Bank”、“Fidelity”、“Coinbase”）。

## 创建账户

1.  前往 **设置 > 账户**。
2.  点击 **添加账户**。
3.  **名称**：给它一个易于识别的名称。
4.  **类型**：
    - **证券**：用于股票、ETF、共同基金。
    - **现金**：用于银行账户或高收益储蓄。
    - **加密货币**：用于加密钱包或交易所。
    - **自定义**：用于房地产、艺术品或其他资产。
5.  **货币**：该账户的主要货币。

## 隐藏账户

您可以“隐藏”一个账户，如果您想保留其数据但将其从主仪表盘总计中移除（例如，为您管理的他人账户）。
